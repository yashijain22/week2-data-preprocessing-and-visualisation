# Week 2 – Data Preprocessing & Visualization

## Titanic Dataset Analysis

This project demonstrates data preprocessing and visualization techniques using the Titanic dataset. The goal of this assignment is to prepare the dataset for machine learning by applying preprocessing steps such as encoding categorical variables and normalizing numerical features.

---

## Objective

The objective of this assignment is to understand how raw datasets can be transformed into a format suitable for machine learning models through preprocessing and exploratory data analysis.

---

## Dataset

The dataset used in this project is the **Titanic Passenger Dataset**, which contains information about passengers such as:

- Passenger Class
- Gender
- Age
- Fare
- Embarkation Port
- Survival Status

Dataset Source:  
https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv

---

## Tasks Performed

The following preprocessing steps were performed:

- Loaded dataset using **Pandas**
- Handled missing values
- Removed unnecessary columns
- Encoded categorical variables
- Normalized numerical features
- Selected relevant features for analysis
- Created multiple visualizations
- Extracted insights from the data

---

## Technologies Used

- Python
- Pandas

- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Files Included

- `week2_titanic_preprocessing_visualization.ipynb` – Jupyter notebook containing preprocessing and visualization steps

---

## Key Insights

- Female passengers had higher survival rates compared to male passengers.
- Passengers in first class had better survival chances than those in lower classes.
- Age and fare distributions show patterns related to survival probability.

---

## Conclusion

This project demonstrates the importance of data preprocessing and visualization in preparing datasets for machine learning. Proper data transformation and analysis help reveal patterns and improve the effectiveness of predictive models.
